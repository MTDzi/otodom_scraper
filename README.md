# Web scrapper for otodom.pl
Made with Scrapy. <br />
### Installation:<br />
`$ git clone https://github.com/pixinixi/otodom_scraper` <br />
`$ cd otodom_scraper` <br/>
`$ pip install -r requirements.txt` <br/>
<br/>
### Usage: <br/>
`$ cd otodom` <br />
`$ scrapy crawl otodomSpider -o [file_name].csv -a city=[city_name]` <br />
<br />
